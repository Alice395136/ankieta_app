# 📊 Analiza Ankiety Powitalnej

Interaktywna aplikacja Streamlit do analizy danych z ankiety.

## 🔧 Uruchom lokalnie

1. Zainstaluj zależności:
```
pip install -r requirements.txt
```

2. Uruchom aplikację:
```
streamlit run app.py
```
